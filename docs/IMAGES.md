# Images / Screenshots

Add screenshots here. Recommended naming:

- `screenshots/settings-ytlite.png` — Skip Silence panel inside YTLite's settings view
- `screenshots/settings-app.png` — Skip Silence bundle in iOS Settings.app
- `screenshots/hud.png` — In-player Smart Speed HUD pill
- `screenshots/architecture.png` — Rendered architecture diagram

When running the extension, you can capture these via:

```bash
# iOS Simulator screenshot (if running in sim)
xcrun simctl io booted screenshot screenshots/settings-ytlite.png

# Or via `screencapture` after AirPlaying the device screen to your Mac
```

The architecture diagram in `docs/ARCHITECTURE.md` is plain ASCII so it does
not need a rendered image.
